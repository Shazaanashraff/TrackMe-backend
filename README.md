# Backend

Express + Mongoose + Socket.IO backend for the bus-tracking system.

## Setup

```bash
npm install
npm run dev
```

## Environment
- Copy .env.example to .env and fill required keys.

## Tests
- Smoke tests: `npm test`
- Planned (see docs): `test:unit`, `test:integration`, `test:ws`
